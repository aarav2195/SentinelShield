# SentinelShield

Advanced Intrusion Detection & Web Protection System

## Overview

SentinelShield is a lightweight Web Application Firewall (WAF) built using Python and Flask. It inspects HTTP requests, detects common web attacks, logs suspicious activities, performs rate limiting, and provides a security dashboard for monitoring detected threats.

## Features

- HTTP Request Inspection
- SQL Injection Detection
- Cross-Site Scripting (XSS) Detection
- Directory Traversal Detection
- Local File Inclusion (LFI) Detection
- Command Injection Detection
- Rate Limiting
- Security Logging
- Dashboard Analytics

## Technologies

- Python
- Flask
- Plotly
- Pandas
- Regex
